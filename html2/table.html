Описание

Элемент

-table-

слуfжит контейнером для элементов, определяющих содержимое таблицы. Любая таблица состоит из строк и ячеек, которые задаются с помощью тегов -tr- и -td-. Внутри -table- допустимо использовать следующие элементы: -caption-, -col-, -colgroup-, -tbody-, -td-, -tfoot-, -th-, -thead- и -tr-

Таблицы с невидимой границей долгое время использовались для верстки веб-страниц, позволяя разделять документ на модульные блоки. Подобный способ применения таблиц нашел воплощение на многих сайтах, пока ему на смену не пришел более современный способ верстки с помощью слоев.
