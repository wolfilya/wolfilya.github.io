 - определяет блок в который помещают предварительно отформатированный текст

Тег: html - является контейнером, который заключает в себе все содержимое веб-страницы, включая теги

Атрибуты - необходимое, существенное свойство объекта

Атрибут тега html lang="en" - Этот атрибут задает язык, на котором написаны все остальные атрибуты данного тега

class="New">Одинарный тег - такие теги добавляют на страницу одиночный объект, и им не нужно для этого заключать в себя какой-то текст.

Тег: head - предназначен для хранения других элементов, цель которых — помочь браузеру в работе с данными.

Вложенный тег meta - определяет метатеги, которые используются для хранения информации предназначенной для браузеров и поисковых систем.

Вложенный тег title - это заголовок html-документа.

Тег Header - задает «шапку» сайта или раздела, в которой обычно располагается

Вложенный тег H1 - представляет собой наиболее важный заголовок первого уровня

Тег: body - предназначен для хранения содержания веб-страницы (контента), отображаемого в окне браузера.

Вложенный тег P - Для определения абзаца в HTML-документе

Тег: footer - задаёт «подвал» сайта или раздела, в нём может располагаться имя автора, дата документа, контактная и правовая информация.
