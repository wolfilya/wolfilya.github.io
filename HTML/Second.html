Атрибут ; с помощью атрибутов можно задать дополнительные параметры для какого-либо тега.

Атрибут тега <html> lang=en - Этот атрибут задает язык, на котором написаны все остальные атрибуты данного тега.

Одинарный тег - такие теги добавляют на страницу одиночный объект, и им не нужно для этого заключать в себя какой-то текст.

Двойной(закрывающий) тег - этот тег указывает,что определенный

Тег: <head> предназначен для хранения других элементов, цель которых — помочь браузеру в работе с данными.

Вложенный тег <meta> - определяет метатеги, которые используются для хранения информации предназначенной для браузеров и поисковых систем.

Вложенный тег <title> - определяет заголовок документа.
